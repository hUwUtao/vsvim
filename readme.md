# vsvim

a monaco alike code editor: stays minimal and embeddable within anything. strip the assyness from vim, where `:q`
is not your major concern.

## story

cef is ass, anything else is bulky. 

nvim is the to-go editor with customizing system.

where, in a game engine or anywhere else, embedding a canvas is so much better than the whole CEF.

## goal

to make an editor that is friendly to even kids

helix is bulky, emacs and nano is ancient ass

no one knows how to press `i` or `:q`

this project will defeat that allegy

make vim great!

> [!WARNING]
> this project is very broken yet!

## how to use

clone this repo to `~/.config/nvim`

- use a mouse-friendly tty such as `windows terminal`, `kitty`, `ghostty`. something like your linux distro tty may supports. nerd-font is out of scope (yet and optional)

we wants to be a traditional and common-sense editor rather than a nerdy one, like vscode

- `ctrl q` to quit, `ctrl s` to save
- `ctrl z` to undo, `ctrl y` to redo
- select and `ctrl c` to copy, `ctrl x` to cut, `ctrl v` to paste
